# Taiwan-news-crawlers
台灣新聞爬蟲

## 簡介

利用 C# .net core 寫的的新聞爬蟲函式庫 

自用、長期維護、緩慢更新


## 開發環境

.net Core 6

## 使用到的其他套件

1. Newtonsoft.Json
2. AngleSharp

## 目前支援爬蟲的新聞網站

1. 中時新聞網
2. 鉅亨網
3. 自由時報網址
4. 中央通訊社
